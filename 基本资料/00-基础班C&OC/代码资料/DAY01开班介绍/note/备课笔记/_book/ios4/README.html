1.自定义动画

  • 在界面上添加图片容器

  • 通过代码关联图片容易, 以便于以后操作

  • 定义容易保存所有图片(序列帧)

    // 1.初始化所有图片
  self.images = [NSMutableArray array];
  for (int i = 1; i <= 36; i++) {
      NSString *imageName = [NSString stringWithFormat:@"img_360car_black_%02d", i];
      UIImage *image = [UIImage imageNamed:imageName];
      [self.images addObject:image];
  }
